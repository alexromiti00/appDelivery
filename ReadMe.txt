Proceso para crear una nueva peticion Http

*Crear nuestras entidades en  "aentities" en la capa de Domain
*Crear nuestro repositorio en *repositories" en la capa de Domain // en esta parte dedinimos los metodos que vamos a utilizar
*Nos pasamos a la capa de Data y creamos nuestro repositorie
*Crear nuestro UseCases
*Creamos nuestro 	Context que nos permitira compartir la informacion entre las distintas pantallas
*Propagar el Context en nuestro navigator
*Compartimos el Api Context en nuestro ViewModel

Proceso para crear un para listar productos desde una categoria(Backend)

*Crear un anueva consulta en nuestro modelo dependiendo que estamos trabajando crear una consulta de FindBy
*Se agrega la consulta en el controlador (getAll)
*Creamos las rutas
*Hacemos pruebas en PostMan si es que queda todo
